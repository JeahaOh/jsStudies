# Event
jQuery를 사용하는게 일반적이므로 Event의 기본만 살펴봄.

## 용어
이벤트 이름 : 이벤트의 이름
이벤트 속성 : 특정 요소(tag)에 이벤트를 직접 연할 때 활용하는 속성
이벤트 리스너 or 핸들러 : 특정 요소(tag)의 이벤트 속성에 연결한 함수
디폴트 이벤트 : 요소(tag)가 가지고 있는 기본적인 이벤트 
            ex) a 태그의 href 속성
이벤트 모델 : 이벤트를 연결하는 방법

## 이벤트의 종류
- 마우스 이벤트
- 키보드 이벤트
- HTML 프레임 이벤트
- HTML 입력 양식 이벤트
- 유저 인터페이스 이벤트
- 구조 변화 이벤트
- 터치 이벤트
