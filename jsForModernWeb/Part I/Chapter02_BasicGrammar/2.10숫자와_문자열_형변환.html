숫자와 문자열 자료형 변환

        다른 자료형을 숫자로 변환할 때는 Number() 함수를 사용함.
        다른 자료형을 문자로 변환할 때는 String() 함수를 사용함.
    
        prompt()함수를 이용해 문자열을 입력받음.
        숫자를 입력해도 문자열로 입력 받음.
        
        code 2-45는 prompt() 함수로 입력받은 문자열 자료형을 숫자 자료형으로 인식하게 함.
    
        숫자가 아닌 값을 입력하게 되면 NaN(; Not a Number)값을 출력 하게됨.
        NaN은 자바스크립트로 나타낼 수 없는 숫자를 의미함.
        문자열을 강제로 숫자형으로 바꿔서 NaN으로 출력 되는 것.
