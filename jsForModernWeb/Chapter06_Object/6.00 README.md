### 객체 요소 접근 I
<객체 이름>.<속성이름>
    -> product.name
    
### 객체 요소 접근 II
<객체 이름>['<속성이름>']
    -> product.['name']

## 이번 장에서 기억해야 하는 용어

- 속성 : 객체가 가진 값
- 메서드 : 객체가 가진 속성 중 함수 자료형인 것
