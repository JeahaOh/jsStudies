ECMAScript5 Array 객체

            Array 확인 method
        Array.isArray()     배열인지 확인
        
        typeof 키워드로 Array객체 자료형을 확인하면 object로 나오므로 isArray()메소드로 확인.
    
 
            Array 탐색 method
        indexOf()          특정 요소를 앞쪽부터 검색
        lastIndexOf()      특정 요소를 뒤쪽부터 검색
        
        내부에 검색하려응 객체가 있으면 해당 객체가 위치하는 인덱스를 리턴,
        없으면 -1리턴
    
   
            Array 반복 method
        forEach()          배열 각각의 요소를 사용해 특정 함수를 for in 반복문처럼 실행
        map()              기존의 배열에 특정 규칙을 적용해 새로운 배열을 생성.
        
        매개변수로 입력 함수에 배열의 요소와 관련된 정보를 넣어 반복.
        for in 반복문과 연관해서 생각하면 됨.
        
        forEach() 메소드의 매개변수
        element     현재 반복에서 배열의 요소
        index       현재 반복에서 요소의index
        array       반복을 수행할 배열의 이름.
    
   
            조건 메소드
        filter()            특정 조건을 만족하는 요소를 추출해 새로운 배열을 만듦.
        every()             배열의 요소가 특정 조건을 모두 만족하는지 확인.
        some()              배열의 요소가 특정 조건을 적어도 하나 만족하는지 확인.
        
        filter() 매개변수로 입력한 함수는 불린타입으로 리턴해야함.
        이때 true인 배열의 요소만 골라서 새로운 배열을 만듦.
        
        every()와 some()
        forEach()나 filter()같은 매개변수를 넣음. filter()와 마찬가지로 불린타입을 리턴함.
        
        every() - 배열의 모든 요소가 true 이면 true.
        some()  - 배열중 하나라도 true 이면 true.
    
            연산 메소드
        reduce()              배열의 요소가 하나가 될 때까지 요소를 왼쪽부터 두 개씩 묶는 함수를 실행
        reduceRight()         배열의 요소가 하나가 될 때까지 요소를 오른쪽부터 두 개씩 묶는 함수를 실행
