Object 객체

        Object : 자바스크립트의 최상위 객체.
        Object 객체의 기본 메소드
        

            
메소드 이름설명

            
constructor()
                객체의 생성자 함수

                
            
hasOwnProperty(name)
                객체가 name 속성이 있는지 확인

                
            
isPrototypeof(object)
                객체가 object의 prototype인지 검사

                
            
propertyIsEnumerable(name)
                반복문으로 열거할 수있는지 확인

                
            
toLocaleString()
                객체를 호스트 환경에 맞는 언어의 문자열로 변환

                
            
toString()
                객체를 문자열로 변환

                
            
valueOf()
                객체의 값을 나타냄
