자료형 구분

        Object 객체 내에 있는 constructor() 메소드는 객체의 생성자 함수를 의미,
        자료형을 검사하는데 유용하게 사용할 수 있다.
        ?
        typeof 연산자로 자료형을 검사하면 문제가 발생할 수 있는 부분이 있음. (code 8-7,8)
        이런 두 대상을 같은 자료형으로 취급 하고 싶을 때는 constructor() 메소드를 사용함.(code 8-9)
        
        
모든 객체에 메서드 추가

        Object는 모든 객체의 최상위 객체임.
        따라서 Object 객체의 prototype에 속성이나 메소드를 추가하면
        모든 객체에서 활용가능함. (code 8-10)
