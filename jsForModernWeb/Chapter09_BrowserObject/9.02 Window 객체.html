Window 객체

        Window 객체는 JavaScript의 최상위 객체임.

새로운 window 객체 생성

        window 객체의 window 생성 method
        open(URL, name, features, replace)    새로운 window 객체를 생성.
        네개의 매개변수는 모두 옵션임.
        URL     : 
        name    : 윈도우간 통신하는 사용하는 윈도우 이름
        feature :
            height     : 새 윈도우의 높이     픽셀값
            width      : 새 윈도우의 너비     픽셀값
            location   : 주소 입력창의 유무   yes, no, 1, 0
            menubar    : -
            resizable  : -
            status     : -
            toolbar    : -
        
        open() 메소드는 새로운 window 객체를 생성하는 메소드
        code 9-4처럼 새로운 윈도우 객체에 접근해 속성과 메소드 사용 가능.

window 객체의 기본 메소드

        moveBy(x,y)    윈도우의 위치를 상대적으로 이동.
        moveTo(x,y)    윈도우의 위치를 절대적으로 이동.
        resizeBy(x,y)  윈도우의 크기를 상대적으로 지정.
        resizeTo(x,y)  윈도우의 크기를 절대적으로 지정.
        scrollBy(x,y)  윈도우 스크롤의 위치를 상대적으로 이동.
        scrollTo(x,y)  윈도우 스크롤의 위치를 절대적으로 이동.
        focus()        윈도우에 초점을 맞춤.
        blur()         윈도우에 맞춘 초첨을 제거.
        close()        윈도우를 닫음.

9.5 Screen 객체

        Screen 객체는 운영체제 화면 속성을 갖는 객체임.
        
        모든 웹 브라우져가 공통으로 갖는 screen 객체 속성
            width           화면 너비
            height          화면 높이
            availWidth      실제 화면에서 사용 가능한 너비
            availHeight     실제 화면에서 사용 가능한 높이
            colorDepth      사용 가능한 색상 수
            pixelDepth      한 픽셀당 비트 수

9.6 Location 객채

        주소 표시줄과 관련, 프로토콜의 종류, 호스트 이름, 문서 위티등의 정보를 갖고 있음
        
            location 객체 속성
        href     문서의 url 주소
        host     host name, port localhost:8888
        hostname 호스트 이름        localhost
        port                     8888
        pathname 디렉토리 경로      /project/jsp
        hash     앵커 이름(#~)     #beta
        search   요청 매개변수      ?param=10
        protocol 프로토콜 종류      http:
        
            localhost 객체의 메소드
        assign(link)    현재 위치로 이동
        reload()        새로고침
        replace(link)   현재 위치를 이동

9.7 Navigator 객체

        웹 페이지를 실행하고 있는 브라우저에 대한 정보가 있음.
            navigator 객체의 속성
        appCodeName     브라우져의 코드 이름
        appName         브라우져의 이름
        appVersion      브라우져 버젼
        platform        사용중인 운영체제의 시스템 환경
        userAgent       웹 브라우져의 전체 정보
