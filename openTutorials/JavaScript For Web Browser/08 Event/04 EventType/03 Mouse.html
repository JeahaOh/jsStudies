이번 예제는 마우스와 관련된 기능을 종합적으로 보여준다.



이벤트 타입

웹 브라우져는 마우스와 관련해서 다양한 이벤트 타입을 지원한다.
- click
- dblclick
- mousedown
- mouseup
- mousemove
- mouseover
- mouseout
- contextmenu




키보드 조합

마우스 이벤트가 호출될 때 특수 키(ctrl, alt, shift)가 눌러진 상태를 감지해야 한다면
이벤트 객체의 프로퍼티를 사용한다.
- event.shiftKey
- event.altKey
- event.ctrlKey




마우스 포인터 위치

마우스 이벤트와 관련한 작업에서는 마우스 포인터의 위치를 알아내는 것이 중요할 때가 있는데,
이런 경우 이벤트 객체의 clientX와 clientY를 사용한다.
